template<typename RandomAccessIterator , typename MapRandomAccessIterator >
void MappedShuffle ( RandomAccessIterator  first,
MapRandomAccessIterator  mapFirst,
size_t  len,
std::function< int(int)>  gen 
)

Rearranges the elements in the range [first,first+len) randomly, assuming that gen is a uniform random number generator. Follows the same algorithm as std::shuffle in C++11 (a Durstenfeld shuffle).

The elements in the range [mapFirst,mapFirst+len) are rearranged according to the same permutation, enabling the permutation to be tracked by the caller.

gen takes an integer n and produces a uniform random output in [0,n).
